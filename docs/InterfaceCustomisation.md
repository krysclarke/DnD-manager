# Interface customisation
## Themes
- ALL included themes must meet WCAG AA, and ideally WCAG AAA, contrast requirements.
- Switchable & customisable 'on-the-fly', all from the main program (separate pop-up window or dialog/card is the preferred interface).
- The web interface defaults to match the main program's theme; however, the ability to use a different theme for the web interface should be considered.
- Display a warning message to the user if the contrast falls below WCAG AA, taking account of the current interface scale

### Included themes
Themes to generate and include:
- D&D 'parchment'
- high-contrast light
- high-contrast dark
- 'arcane'
- purple
	- darkest color #1d1433

## UI sizing
- Allow the UI scale to be increased/decreased by button click and a slider (0.5x - 5.0x, in 0.5x steps).
- Allow Web interface to be scaled separately from the main program.
